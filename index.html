Serverless image analysis powered by Amazon Rekognition.

Architecture Overview

This platform leverages a purely event-driven architecture on AWS and uploads to S3 trigger Lambda functions which orchestrate computer vision analysis and metadata storage in DynamoDB.

Supported Formats: Please ensure all uploads are standard .JPG files.

Stack

  • AWS S3 & Lambda
  • Amazon Rekognition
  • DynamoDB
  • API Gateway
